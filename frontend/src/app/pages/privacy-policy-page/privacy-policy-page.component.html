Types of Data That May Be Collected

While playing online games on this website, OnlineCheckers.org may collect the following information:

  • Game Data: Player nicknames, game moves, and game statistics (pieces count, game status).
  • Chat Messages: Messages sent during gameplay between players.

How collected data is used

OnlineCheckers.org uses the collected data for the following purposes:

  • To create and manage game sessions between players
  • To maintain game state and move history during a game session
  • To facilitate in-game chat between players

Data Storage and Retention

Games data is stored in AWS RDS (Relational Database Service).

Game Data Retention: Game sessions and associated data (nicknames, moves, chat messages) are deleted after 30 days.

Moderation Data: Statistics about inappropriate nickname attempts may be retained for up to 1 year. This data is used solely to enhance content filtering and prevent abuse.

  • Deletion: Request deletion of your personal data
